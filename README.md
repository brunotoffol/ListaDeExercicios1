# Lista De Exercícios Lógica de Programação - Utilizando a Linguagem C#

Nesta lista de 23 exercícios foram praticados conceitos da linguagem de programação C#, como laços de repetição, estrutura condicionais, concatenação de variáveis, etc.

### Como utilizar:
1. Clone o repositório ou baixe o código fonte
2. Abre o terminal ou prompt de comando e navegue até a pasta raiz
3. Utilize o comando abaixo para restaurar as depências do projeto

```
dotnet restore
```
4. Em seguida, compile a solução utilizando o comando:
```
dotnet build --configuration Release
```
5. Para Executar o projeto compilando em tempo real
```
dotnet run --project ListaDeExercicios1
```
6. Para executar o arquivo compilado, navegue até a pasta ./ListaDeExercicios1/bin/Release/net8.0/ e execute o arquivo:
```
ListaDeExercicios1.exe
```

## Requisitos

- .NET SDK (recomendado .NET 8.0 ou superior) para compilação e execução do projeto.
